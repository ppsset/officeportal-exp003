# WAG Portal

Static prototype for the MEP design office portal optimized for desktop and mobile. Open `index.html` in a browser to view the experience.

## Features
- Collapsible sidebar with shortcuts to Office Updates, Employee Hub, Program Calendar, Tech Knowledge, Shared Facilities, and Wellbeing modules.
- Home dashboard showing key counts, module shortcuts, and a recent activity feed.
- Module tabs with grid and list layouts for news, policies, polls, requests, calendar items, reservations, and knowledge posts.
- Facility and knowledge calendar views for quick scheduling context.
- Wellbeing monitoring cards with live-updating air quality, noise, illumination, and power readings across floors.
- Lightweight sign-in modal for users and admins with status badges.
